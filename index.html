3D Print Result

  • Use PLA as the 3D printing materials and print in Makerplace in Langford B

  • At first, the printing is not start from the connection point which made the final result would be just multiple line segment without connection. With the help of makerplace staff, we changed the printing order to make sure it starts from the point. This change solved the previous issue but also increase the printing time from 1 hr to 6 hr

Summary

  • The final product will move as I expected. However, using PLA to print makes the piler sturdier which caused it a little difficult to use
